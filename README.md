#### Setup
- Use NodeJS 18 for this project.

On both `blog` and `server` directories, install dependencies:
```sh
npm install
```

#### Local development
On `blog` you can start the dev server using the following:
```sh
npx ng start
```
On `server` you can start it by doing:
```sh
npm run dev
```
